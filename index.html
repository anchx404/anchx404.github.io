Long run relationship between unemployment and inflation
  • Stata
  • ARDL
key findings:
The study found that the Phillips curve theory, which suggests an inverse relationship between unemployment and inflation, remains relevant in the 21st century based on data collected from a government website and analyzed using the autoregressive distributed lag (ARDL) approach.
problem: Investigate the relationship between unemployment and inflation in the 21st century. The purpose of this is to identify if it is still relevant for today's world.
solution: Collected data from the government website, processed it, and analyze their relationship using ARDL.
result: It was found that Phillips curve is still relevant as a theory.
